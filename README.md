CoreWindow
==========

# General
- Cross Platform library for creating OpenGL window.
- include 1header, add 1implementation.
- OpenGL用ウインドウ作成クロスプラットフォームライブラリ.
- 1ヘッダーのインクルード，1実装ファイルの追加で利用可能.

# Platform
- Window
- MacOSX(Cocoa)
- Linux(X11)
